# SecureProfile

A modern, responsive personal cybersecurity portfolio website developed for **Project 1** of the **Cybrexa Cybersecurity Internship Program**.

## 📖 About

SecureProfile is a cybersecurity portfolio website that showcases my learning journey, technical skills, and career goals in cybersecurity.

## ✨ Features

- Responsive Design
- Modern Dark UI
- Hero Section
- About Page
- Skills Page
- Cybersecurity Roadmap
- Contact Form with Validation
- GitHub & LinkedIn Links
- Mobile Friendly

- JavaScript
- Git
- GitHub
